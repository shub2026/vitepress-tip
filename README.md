## VitePress 模板使用手册

这是一个基于 [VitePress](https://vitepress.dev/) 的中文教程网站模板。
它以简约风格设计。无论你是程序员、写手还是知识工作者，都可以用它快速搭建一个专业的知识分享网站。

**关键特性：**
- 📝 完全基于 Markdown，自动生成网站
- ⚡ 超快的开发和构建速度
- 🎨 干净美观的风格设计
- 🔄 实时预览，改完立即看到效果
- 📱 自适应手机和电脑屏幕
