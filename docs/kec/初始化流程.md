# 初始化流程
```sh
# 1. 安装依赖
npm install

# 2. 创建数据库并执行迁移
npx prisma migrate dev

# 3. 生成Prisma客户端
npx prisma generate

# 4. 创建超级管理员（可选，首次部署时执行）
npm run db:seed
```
## 建议的部署文档
对于新环境部署（本地开发或生产环境），流程应该是：
1. 首次部署：
- 执行数据库迁移创建表结构
- 运行 seed.js 创建超级管理员
- 使用超级管理员登录
- 通过 Excel 导入功能导入基础数据（培养层次、学院、专业、课程等）
2. 后续启动：
- 直接启动服务即可
- 不需要再次执行 seed

## 完整的部署流程
```sh
# 1. 克隆项目
git clone https://github.com/shub2026/kec-manager.git
cd kec-manager

# 2. 安装依赖
npm install
cd server && npm install && cd ..
cd client && npm install && cd ..

# 3. 初始化数据库（创建表结构）
cd server
npx prisma migrate dev
npx prisma generate

# 4. 创建超级管理员（首次部署必需）
npm run db:seed

# 5. 启动服务
cd ..
npm run dev
```
**默认账号**
- 用户名: admin
- 密码: admin@123456
- 角色: super_admin
- ⚠️ 首次登录后请立即修改
