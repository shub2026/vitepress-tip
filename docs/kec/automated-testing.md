## CI/CD 流水线

`.github/workflows/test.yml` 配置：

```yaml
触发条件: push/PR 到 main、develop 分支
并行任务:
  - test (Node 18.x, 20.x)
    ├─ 安装依赖
    ├─ Prisma generate
    ├─ 运行测试 + 生成覆盖率报告
    └─ 上传 Codecov
  - lint
    └─ npm audit 安全检查（moderate 级别）
  - build-test
    └─ 前端构建验证
```

---

## 运行测试

```bash
# 进入服务端目录
cd server

# 运行所有测试
npm run test:run

# 生成覆盖率报告
npm run test:coverage

# 监视模式（开发时推荐）
npm test

# 运行指定测试文件
npx vitest tests/auth.test.js

# 详细输出
npx vitest --reporter=verbose
```
