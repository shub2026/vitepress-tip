## 版本说明

- **主版本号** (v1.x.x)：不兼容的 API 修改
- **次版本号** (vx.1.x)：新功能（向后兼容）
- **修订号** (vx.x.1)：Bug 修复（向后兼容）

## 发布流程

1. 更新版本号：`npm run version:patch`
2. 在此文件中记录变更内容
3. 提交代码并打标签：`git tag v1.5.19`
4. 推送标签：`git push && git push --tags`
