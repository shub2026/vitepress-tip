# Lightroom 预设使用技巧指南

---

## 一、理解预设的本质

预设不是"一键出片"的魔法，而是**一组参数起点的快照**。它帮你跳过重复的基础调整，把时间留给对具体照片的微调。

核心认知：

- 预设记录的是滑块位置，不是最终效果
- 同一预设用在不同曝光、色温、对比度的照片上，结果差异巨大
- 好的工作流 = 预设起步 + 逐张微调 + 偶尔惊喜

---

## 二、预设的分类思维

### 2.1 按用途分

| 类型 | 作用 | 例子 |
|------|------|------|
| **基础校正型** | 统一白平衡、曝光基准、镜头校正 | "标准校准"、"自动曝光修正" |
| **风格化型** | 赋予整体色调氛围 | "胶片模拟"、"电影色调"、"日系清新" |
| **场景专用型** | 针对特定拍摄场景 | "逆光人像"、"城市夜景"、"黄金时刻" |
| **单步调整型** | 只调一个维度，可叠加组合 | "提亮肤色"、"压暗天空"、"加颗粒" |

> **建议**：你的预设库应该以"基础校正型 + 单步调整型"为骨架，"风格化型"按需积累，"场景专用型"精而不多。

### 2.2 按来源分

- **官方预设**：Lightroom 自带，质量稳定，适合学习拆解
- **自制预设**：从自己最好的作品中提取，最贴合你的风格
- **第三方预设**：购买或免费获取，质量参差，需筛选和适配

## 三、自制预设的方法论

### 3.1 从最佳作品反向提取

不要凭空造预设，而是：

1. 回顾你修过的最满意的照片（5-10 张风格统一的）
2. 找到它们的共同调整特征：色调曲线形状、色彩分离参数、HSL 偏移方向
3. 选一张最具代表性的，以此为基础保存预设
4. 在其他照片上测试，逐步收敛参数

### 3.2 保存预设时勾选什么

这是最容易被忽略的关键步骤——**不是所有参数都该写进预设**：

| ✅ 建议勾选 | ❌ 建议不勾选 |
|------------|-------------|
| 白平衡（风格化预设） | 裁剪 |
| 基础曝光微调 | 污点修复 |
| 色调曲线 | 局部调整（画笔/渐变/径向） |
| HSL | 锐化（视具体照片而定） |
| 色彩分离 | 降噪（视 ISO 而定） |
| 暗角/颗粒 | 镜头校正（应单独处理） |

> **原则**：预设只固化"风格意图"，不固化"技术校正"。裁剪和局部调整永远留给具体照片。

### 3.3 分层预设策略

把预设拆成"基础层 + 风格层 + 点缀层"，每次组合使用：

```
基础层：白平衡归位 + 曝光基准 + 镜头校正
  ↓ 叠加
风格层：色调曲线 + HSL + 色彩分离
  ↓ 叠加
点缀层：暗角 + 颗粒 + 单项微调
```

**操作方法**：先应用基础层预设，再应用风格层预设（取消勾选基础层已有的参数），最后点缀层。这样换风格只需替换中间层，不动基础。

## 四、预设的正确使用姿势

### 4.1 应用预设后必须做的事

预设应用后，按以下清单逐项检查并微调：

1. **白平衡**：预设可能改变色温，检查是否符合当前照片
2. **曝光**：预设的曝光偏移未必适合这张，看直方图调整
3. **高光/阴影**：不同动态范围的照片，高光阴影恢复量不同
4. **HSL 中的橙色**：肤色是否自然，这是人像的必检项
5. **色调曲线**：预设曲线可能让暗部死黑或高光溢出，按 `J` 检查 clipping
6. **整体观感**：缩小到适合屏幕看全图，不要只盯细节

### 4.2 预设强度控制

Lightroom 没有直接的预设不透明度滑块，但有变通方法：

**方法一：选择性勾选**
- 应用预设时只勾选部分参数，降低预设对画面的影响面

**方法二：手动回拉**
- 预设把色调曲线拉到极端？手动拉回 30-50%
- 预设加了浓重色彩分离？把饱和度砍半
- 相当于给预设做"减法"

**方法三：快照对比法**
- 应用预设前先建一个快照（Snapshot）："原片"
- 应用预设后再建一个："预设版"
- 在两个快照间切换，找到你真正想要的中间状态
- 在原片基础上手动还原那个中间状态

**方法四：Lightroom 的预设量滑块（2023.6+ 版本）**
- 新版 Lightroom 已支持 Preset Amount 滑块
- 应用预设后，面板顶部出现量控制条，0-100% 调整
- 这是官方的预设强度控制方案

### 4.3 预设不适用的情况

以下场景慎用或不用预设：

- **严重欠曝/过曝**：预设基于正常曝光设计，极端曝光下参数会失控
- **混合光源**：预设的白平衡偏移会和现场色温打架
- **高 ISO 照片**：预设的锐化/降噪参数不匹配，可能放大噪点
- **已经很有氛围的原片**：有时候原片已经很好，预设反而是画蛇添足

## 五、预设管理技巧

### 5.1 文件夹结构

```
预设根目录/
├── 00-基础校正/        ← 白平衡归位、曝光基准
├── 01-风格-胶片/       ← 各类胶片模拟
├── 02-风格-电影/       ← 电影色调
├── 03-风格-清新/       ← 日系、明亮风
├── 04-场景-人像/       ← 逆光人像、影棚人像
├── 05-场景-风光/       ← 日落、夜景
├── 06-单步调整/        ← 提亮肤色、加颗粒等
└── 99-实验/            ← 测试中的预设，不稳定
```

- 用数字前缀控制排序，最常用的排最前
- 超过 30 个预设时必须分文件夹，否则选择疲劳
- 定期清理：3 个月没用过的移入"归档"或删除

### 5.2 命名规范

好的预设名应包含：**风格 + 特征 + 变体**

```
✅ 好的命名：
  Film-KodakPortra400-暖调
  Cinematic-TealOrange-强对比
  Portrait-柔光-肤色提亮

❌ 差的命名：
  我的预设3
  最终版
  调色1
```

### 5.3 导入导出与备份

- **导出**：右键预设文件夹 → Export，生成 `.xmp` 文件
- **导入**：预设面板 `+` → Import Presets，选择 `.xmp` 文件
- **备份**：预设文件实际存储位置——
  - Windows：`C:\Users\用户名\AppData\Roaming\Adobe\CameraRaw\Settings\`
  - macOS：`~/Library/Application Support/Adobe/CameraRaw/Settings/`
- 定期把整个 Settings 文件夹备份到云盘

## 六、第三方预设的适配

### 6.1 为什么第三方预设在你的照片上不好看

常见原因：

1. **RAW 格式差异**：不同厂商的 RAW 色彩起点不同（佳能偏暖、索尼偏绿、富士自带胶片模拟）
2. **曝光差异**：预设作者的照片曝光和你不同，同样的参数偏移量效果不一样
3. **光线条件差异**：预设是在特定光线下调的，换到另一种光线全变味
4. **版本差异**：旧版 LR 的预设在新版上可能表现不同

### 6.2 适配步骤

1. 先确保照片白平衡和曝光基本正常，再应用预设
2. 应用后，优先调整白平衡和曝光，让画面回到合理区间
3. 检查色调曲线是否过度——最常见的问题是第三方预设曲线太猛
4. HSL 中肤色是否自然——优先修橙色
5. 色彩分离饱和度通常需要砍半
6. 如果整体偏差太大，不如把预设当参考，手动从零调

### 6.3 拆解学习第三方预设

最高效的学习方式不是"用"预设，而是"拆"预设：

1. 应用预设后，逐面板查看每个滑块的位置
2. 问自己：这个调整的意图是什么？
3. 关掉该参数，对比前后差异
4. 记住你喜欢的调整手法，融入自己的预设体系

重点关注：
- **色调曲线的形状**——这是风格的骨架
- **色彩分离的色相选择**——这是氛围的来源
- **HSL 中的调整方向**——这是色彩偏好的密码

## 七、高级技巧

### 7.1 预设 + 蒙版 AI 的组合拳

1. 应用整体风格预设
2. 用"选择主体"蒙版，单独给主体应用另一个预设（如提亮/柔化）
3. 用"选择天空"蒙版，单独给天空应用压暗/加蓝预设
4. 相当于对同一张照片的不同区域使用不同风格

### 7.2 预设 + 智能集合成工作流

在 Smart Collections 中设置条件：

- 评分 ≥ 3 + 关键词包含"人像" → 自动归集到"待处理人像"
- 应用对应的"人像基础"预设后，移到下一步 Smart Collection
- 适合大批量工作流的半自动化处理

### 7.3 预设的版本迭代

预设不是一次定型的，应该随你的审美进化：

```
v1.0 初版 → 5 张照片测试 → 发现肤色偏黄
v1.1 橙色色相 +3 → 10 张测试 → 暗部不够透
v1.2 阴影 +10 → 稳定使用 2 周 → 想尝试更大胆的风格
v2.0 重调色调曲线 → ...
```

- 在预设名后加版本号：`Film-Portra-v1.2`
- 保留旧版本，风格转换时可以回溯
- 每次迭代只改 1-2 个参数，明确改了什么

### 7.4 预设与 Lightroom 云端同步

- Lightroom（云版）的预设会自动通过 Adobe Cloud 同步到手机端
- Lightroom Classic 需手动开启：`Edit → Preferences → Lightroom Sync → Sync Presets`
- 移动端修图时可直接使用桌面端创建的预设
- 适合在平板上做初筛和初调，回桌面精修

## 八、预设使用的常见误区

| 误区 | 正解 |
|------|------|
| 预设能拯救烂片 | 预设只能锦上添花，前期拍好是根本 |
| 预设越多越好 | 30 个精选 > 300 个未整理 |
| 每张照片都要用预设 | 好原片直接微调可能更佳 |
| 同一预设适合所有照片 | 不同场景需要不同起点，至少微调白平衡和曝光 |
| 付费预设一定比免费好 | 核心是参数理解，不是价格标签 |
| 预设 = 最终效果 | 预设是起点，微调才是完成 |
