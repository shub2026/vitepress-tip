## 一、灯光核心原则

### 1.1 证件照灯光四大要求

| 要求 | 说明 | 常见后果 |
|------|------|----------|
| **均匀无阴影** | 面部受光均匀，无明显阴影区块 | 阴阳脸、鼻影过重 |
| **色温一致** | 主光、辅光、背景光色温统一 | 肤色偏色 |
| **眼神光明亮** | 眼睛中有清晰、居中的眼神光 | 眼神呆滞 |
| **背景纯净** | 背景受光均匀，与主体分离 | 背景不均、泛灰 |

### 1.2 光位基础术语

```
           主光 (Key Light)
                ↓
                ● 45°
               ╱
              ╱
             ╱
            ╱
           ╱
    ──────●──────── 相机 (Camera)
          ↑
       辅光/反光板
```

- **主光 (Key Light)**：主要塑形光源，决定整体光影基调
- **辅光 (Fill Light)**：补充阴影，降低光比
- **轮廓光 (Rim Light)**：勾勒人物边缘，分离主体与背景
- **背景光 (Background Light)**：照亮背景，营造层次

### 1.3 光比控制

| 光比 | 效果 | 适用场景 |
|------|------|----------|
| 1:1 | 面部完全均匀，无立体感 | 入门级证件照 |
| 1:2 | 轻微阴影，保留立体感 | 标准证件照 |
| 1:3 | 明显立体感 | 形象照、简历照 |

> **光比计算**：主光亮度 ÷ 辅光亮度。例如主光1/1、辅光1/2时，光比为 1:2。

## 二、闪光灯 vs 持续光

### 2.1 核心对比

| 特性 | 闪光灯 | 持续光 (LED/常亮灯) |
|------|--------|---------------------|
| **光线可见性** | 所见非所得，需试拍 | WYSIWYG，所见即所得 |
| **凝固运动** | 优秀（闪光持续时间可达1/10000s） | 受快门速度限制 |
| **功率输出** | 强（400-1200Ws） | 较弱（60-400W） |
| **光质控制** | 精确可调 | 直观易控 |
| **学习曲线** | 较陡 | 较缓 |
| **适用场景** | 证件照、产品、人像 | 视频混合、直播、入门 |
| **发热量** | 低（闪光瞬间） | 高（持续发光） |

### 2.2 何时选择闪光灯

- ✅ 需要最低 ISO（100-200）以获得最佳画质
- ✅ 需要使用大功率柔光箱
- ✅ 长时间连续拍摄（不产生持续热量）
- ✅ 需要凝固人物表情瞬间
- ✅ 主体与强环境光对抗

### 2.3 何时选择持续光

- ✅ 初学者布光练习
- ✅ 需要同时拍摄视频
- ✅ 对着镜子实时调整光线
- ✅ 引导非专业模特放松表情

### 2.4 设备推荐层级

| 层级 | 闪光灯推荐 | 持续光推荐 |
|------|-----------|-----------|
| 入门 | 机顶闪光灯（TTL） | LED面板灯 |
| 进阶 | 外拍灯（300-400Ws） | 棒灯/聚光灯 |
| 专业 | 影室闪光灯（400-600Ws） | 专业LED聚光灯 |

## 三、相机参数基础设置

### 3.1 闪光灯同步速度

> ⚠️ **重要**：使用闪光灯时，快门速度必须低于同步速度，否则画面会出现黑色条纹。

**各品牌相机同步速度参考：**

| 相机品牌 | 机型示例 | 机械快门同步速度 | 电子前帘同步速度 |
|----------|----------|------------------|------------------|
| Canon | EOS R5/R6 | 1/200s | 1/250s |
| Nikon | Z6 III / Z8 | 1/200s | 1/250s |
| Sony | A7 IV | 1/200s | 1/250s |

**高速同步 (HSS/FP Sync)**：
- 允许使用高于同步速度的快门（如1/1000s、1/2000s）
- 会损失约1-2档闪光功率
- 适用于户外强光环境或需要大光圈虚化

### 3.2 推荐参数组合

| 场景 | 光圈 | 快门 | ISO | 白平衡 |
|------|------|------|-----|--------|
| 影室闪光灯 | f/8 - f/11 | ≤1/200s | 100 | 5600K |
| 外拍灯 | f/5.6 - f/8 | ≤1/250s | 100-200 | 5600K |
| 机顶闪光灯 | f/5.6 - f/8 | ≤1/200s | 200-400 | 5600K |
| LED持续光 | f/4 - f/5.6 | 1/60s | 400-800 | 5600K |

### 3.3 测光模式选择

- **中央重点测光**：对人物面部测光，确保肤色正确
- **点测光**：精确控制眼神光区域
- 避免使用矩阵/评价测光（环境光权重过大）

### 3.4 镜头焦距建议

| 焦距 | 适用场景 | 说明 |
|------|----------|------|
| 50mm | 全画幅入门 | 轻微畸变 |
| 85mm | 人像黄金焦段 | 畸变最小，立体感强 |
| 70-110mm | 证件照推荐范围 | 专业影楼常用 |

> 焦距过短（如35mm）会造成面部变形（鼻子显大）；焦距过长（如135mm+）会压缩背景，缺乏空间感。

## 四、入门级布光方案

> 适合：预算有限、刚接触摄影布光的爱好者

### 4.1 方案 A：自然光 + 反光板

#### 设备清单

| 设备 | 推荐规格 | 大致价位 | 必选 |
|------|----------|----------|------|
| 相机 | APS-C或全画幅微单/单反 | 自备 | ✅ |
| 镜头 | 50mm或85mm定焦 | 1000-3000元 | ✅ |
| 反光板 | 五合一反光板（110cm） | 80-200元 | ✅ |
| 三脚架 | 稳定支撑 | 200-500元 | 建议 |
| 背景布 | 白色/灰色背景布 | 50-100元 | ✅ |

#### 布光示意图

```
                    自然光（窗户）
                         ↓
                        ╲
                         ╲ 45°
                          ╲
                           ╲
                            ●──────── 被摄者
                            │
                            │
                      [反光板] ← 45° 补光
                            │
                            │
                         [相机]
```

**位置参数：**
- 窗户光：从侧面45°角照射
- 反光板：位于相机侧对面，距离被摄者约0.5-1米
- 相机：正对被摄者双眼水平高度

#### 相机参数设置

```yaml
拍摄模式: Manual (M手动模式)
光圈: f/5.6 - f/8
快门: 1/125s - 1/200s（视窗光强度调整）
ISO: 200 - 400
白平衡: 5600K 或 "日光" 模式
对焦: 中央单点对焦，锁定眼睛
```

#### 操作步骤

1. **选择时间**：选择晴天的上午10点前或下午3点后，避免正午强光
2. **寻找位置**：靠近窗户，被摄者与窗户呈45°角
3. **放置反光板**：在被摄者背光侧，用白色面朝向被摄者
4. **测试曝光**：试拍并检查直方图，避免高光溢出
5. **调整光比**：反光板越近，阴影越浅
6. **拍摄**：保持被摄者表情自然，眼睛看向镜头

#### 常见问题与注意事项

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 阴阳脸 | 窗户光过强过硬 | 用白色薄布遮挡部分光线柔化 |
| 眼神无光 | 反光板位置不当 | 将反光板略低于眼睛高度 |
| 背景泛蓝 | 天空色温影响 | 关闭窗帘改用室内白墙反射光 |

### 4.2 方案 B：机顶闪光灯跳闪

#### 设备清单

| 设备 | 推荐规格 | 大致价位 | 必选 |
|------|----------|----------|------|
| 相机 | 带热靴的单反/微单 | 自备 | ✅ |
| 机顶闪光灯 | 支持TTL的闪光灯 | 1000-3000元 | ✅ |
| 反光板 | 五合一反光板（110cm） | 80-200元 | ✅ |
| 引闪器（可选） | 用于离机闪 | 200-500元 | 建议 |

#### 跳闪布光原理

```
        天花板/白墙
            ↓↑
           反射
            │
        [闪光灯]──→ ──→ ──→ [漫射光线] → 柔和均匀
            │
            ↓
        闪光 → 硬光（不采用）
```

**跳闪**：将闪光灯朝向天花板或白色墙面，利用反射光柔化光线。

#### 布光示意图

```
                 天花板/白墙
                      ↕ 反射
                       │
            ┌──────────┴──────────┐
            │                     │
            │    漫射柔和光线      │
            │    ↓ ↓ ↓            │
            │                     │
            └───[机顶闪·朝上]─────┘
                       │
                    ●───→ 柔和均匀光照 → 被摄者
                   ╱
                  ╱
             [反光板] ← 补眼神光
                  │
               [相机]
```

**位置参数：**
- 闪光灯：垂直向上或略向后倾斜（45°内）
- 反光板：放置于被摄者前方，略低于眼睛
- 距离：闪光灯到天花板约2-3米

#### 相机参数设置

```yaml
拍摄模式: Manual (M) 或 Aperture Priority (Av)
光圈: f/5.6 - f/8
快门: ≤1/200s（机械快门）
ISO: 200 - 400
闪光灯模式: TTL 或 Manual (1/8 - 1/4功率)
闪光灯方向: 朝上/朝后跳闪
白平衡: 5600K
```

#### 操作步骤

1. **安装闪光灯**：将机顶闪光灯安装到相机热靴
2. **设置跳闪**：旋转闪光灯灯头，朝向天花板或侧面白墙
3. **放置反光板**：在被摄者正前方下方，补足眼神光
4. **测试拍摄**：
   - 先试拍一张，检查曝光
   - 如过曝，减少闪光灯功率或增大光圈
   - 如欠曝，增加闪光灯功率或提高ISO
5. **微调**：观察面部阴影，适当调整反光板位置

#### 常见问题与注意事项

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 跳闪无效 | 天花板/墙面非白色 | 改用白色反光伞或柔光箱 |
| 光线偏暖 | 暖色墙面反射 | 使用RAW格式拍摄，后期校正 |
| 闪光不同步 | 快门速度超过同步速度 | 降低快门速度至1/200s或更低 |
| 红眼 | 闪光直射眼睛 | 确保跳闪角度够大，光线经反射到达人眼 |

## 五、进阶级布光方案

> 适合：有一定摄影基础，希望提升成片质量的用户

### 5.1 双灯布光：主光 + 辅光

#### 设备清单

| 设备 | 推荐规格 | 大致价位 | 必选 |
|------|----------|----------|------|
| 相机 | APS-C或全画幅微单/单反 | 自备 | ✅ |
| 外拍灯/闪光灯 | 200-400Ws，支持离机 | 1500-5000元/盏 | ✅×2 |
| 柔光箱/反光伞 | 90-120cm八角柔光箱 | 200-500元/个 | ✅ |
| 灯架 | 2.5-3米高度 | 150-300元/个 | ✅×2 |
| 引闪器 | 无线引闪，支持TTL | 300-800元 | ✅ |
| 背景布 | 白色无缝背景纸 | 100-300元 | ✅ |
| 反光板 | 五合一反光板 | 80-200元 | 建议 |

#### 布光示意图

```
    主灯 (Key Light)              辅灯 (Fill Light)
         ↓                            ↓
         ● 45°                      ● 45°(对侧)
        ╱                            ╱
       ╱                            ╱
      ╱                            ╱
     ╱                            ╱
    ╱                            ╱
   ●────────────────────────────●  ← 被摄者
         ↓                            ↓
      柔光箱                       柔光箱
   (主光光源)                   (辅光光源)
                                          │
                                    [相机] ← 正对被摄者
```

**位置参数：**

| 灯光 | 角度 | 高度 | 距离被摄者 | 功率比 |
|------|------|------|------------|--------|
| 主灯 | 45°侧光 | 与眼睛平齐或略高 | 1.5-2米 | 1/1（全光） |
| 辅灯 | -45°侧光（对侧） | 与眼睛平齐 | 1.5-2米 | 1/2 - 1/4 |

#### 相机参数设置

```yaml
拍摄模式: Manual (M)
光圈: f/8
快门: ≤1/200s
ISO: 100
白平衡: 5600K (日光)
测光模式: 中央重点测光
照片格式: RAW + JPEG
```

#### 操作步骤

1. **搭建灯架**：将两盏灯分别置于被摄者两侧
2. **安装柔光箱**：
   - 主灯柔光箱朝向被摄者
   - 辅灯柔光箱朝向被摄者
3. **设置主灯**：功率设为全光（1/1），距离约1.5米
4. **设置辅灯**：功率设为1/2，用白色反光板测试光比
5. **测试光比**：观察面部阴影，主光侧阴影略深但不过重
6. **调整背景光**（可选）：在背景灯前加柔光箱，照亮背景
7. **拍摄**：多拍几张，检查直方图

#### 光比调节技巧

| 期望效果 | 主光:辅光比例 | 阴影程度 |
|----------|---------------|----------|
| 平面均匀 | 1:1 | 无阴影 |
| 轻微立体 | 2:1 | 轻阴影 |
| 标准立体 | 3:1 | 中等阴影 |
| 戏剧效果 | 4:1 | 深阴影 |

> **证件照推荐**：光比2:1至3:1，保留自然立体感的同时避免阴阳脸。

#### 常见问题与注意事项

| 问题 | 原因 | 解决方法 |
|------|------|----------|
| 两侧亮度不一致 | 灯架高度或功率不一致 | 调整灯架高度或功率 |
| 鼻影过重 | 主光位置过低或光质过硬 | 提高主灯位置，改用更大柔光箱 |
| 背景过暗 | 背景光不足 | 添加第三盏背景灯 |
| 光线发硬 | 柔光箱尺寸不够大 | 增加柔光箱尺寸或使用八角柔光箱 |

### 5.2 鳄鱼光（证件照经典布光）

> 鳄鱼光是证件照行业的标准布光，上下一对灯光形成均匀眼神光。

#### 布光示意图

```
                    八角柔光箱 (主光)
                         ↓
                        45°
                         │
                         ▼
    ○ ← 眼神光 ← ─ ─ ─ ● ─ ─ ─ → ○ ← 眼神光
              ↑                 ↑
         长条柔光箱(辅光)  长条柔光箱(辅光)
              ↓45°            ↓45°
              │               │
              └───────┬───────┘
                      │
                      ▼
                  [被摄者]
                      │
                 [相机] ← 正对
```

**位置参数：**

| 灯光 | 附件 | 角度 | 高度 | 位置 |
|------|------|------|------|------|
| 主灯 | 八角柔光箱(90-120cm) | 45°俯角 | 高于眼睛 | 正上方略前 |
| 辅灯 | 长条柔光箱/雷达罩 | 45°仰角 | 低于眼睛 | 正下方 |

**特点**：
- 主灯打亮面部轮廓
- 辅灯消除颈部和下颚阴影
- 形成上下双眼神光，眼睛更有神

#### 操作步骤

1. 主灯置于被摄者正前方上方45°
2. 辅灯置于被摄者正前方下方45°（用长条柔光箱）
3. 两灯功率比约1:2（主灯:辅光）
4. 观察眼睛中是否形成上下两个眼神光
5. 调整辅灯角度，确保不产生下颚阴影

## 六、专业级布光方案

> 适合：影楼、工作室、专业摄影师

### 6.1 三灯布光：主光 + 辅光 + 轮廓光

#### 设备清单

| 设备 | 推荐规格 | 大致价位 |
|------|----------|----------|
| 相机 | 全画幅专业微单/单反 | 自备 |
| 影室闪光灯 | 400-600Ws ×3 | 3000-8000元/盏 |
| 八角柔光箱 | 120-150cm | 400-800元 |
| 条形柔光箱 | 60-90cm | 200-400元 |
| 标准罩/蜂巢 | 配轮廓光使用 | 100-200元 |
| 灯架 | 3米灯架 ×3 | 300-500元/个 |
| 引闪器 | 专业无线引闪系统 | 1000-2000元 |
| 背景纸 | 白色无缝卷纸 | 200-500元 |

#### 布光示意图

```
                 轮廓光 (Rim Light)
                        ↓
                        ●
                        │ ← 从后方高处打向肩部
                        │
    主灯 (Key)    辅灯 (Fill)    背景光 (BG)
        ↓              ↓            ↓
        ● 45°         ● 45°         ●
       ╱            ╱            ╱
      ╱            ╱            ╱
     ╱            ╱            ╱
    ╱            ╱            ╱
   ●────────────────────────────● ← 被摄者
                                        │
                                   [相机] ← 正对
```

**位置参数：**

| 灯光 | 附件 | 角度 | 高度 | 距离 | 功率 |

1. **搭建背景**：安装白色背景纸，确保平整无褶皱

## 八、特殊肤色与发色调整

### 8.1 不同肤色的灯光策略

#### 白皙皮肤

| 问题 | 原因 | 调整方法 |
|------|------|----------|
| 容易过曝 | 皮肤反光率高 | 降低主光功率10-15%，略微欠曝 |
| 质感丢失 | 柔光箱过柔 | 使用雷达罩保留皮肤细节 |
| 立体感弱 | 光比过低 | 适当提高光比至3:1 |

#### 中等肤色

| 问题 | 原因 | 调整方法 |
|------|------|----------|
| 肤色偏黄 | 主光源色温偏低 | 提高色温至5600-5800K |
| 阴影过重 | 光比过高 | 增强辅光，降低光比至2:1 |

#### 深色/小麦肤色

| 问题 | 原因 | 调整方法 |
|------|------|----------|
| 暗部死黑 | 皮肤吸光性强 | 大幅增强辅光，光比降至1.5:1 |
| 立体感不足 | 皮肤漫反射强 | 使用轮廓光增强边缘分离 |
| 肤色沉闷 | 反光不足 | 使用银面反光板近距离补光 |

#### 技巧要点

> **深色皮肤需要更多光线量**。深色皮肤会吸收更多光线，因此需要：
> - 更高的主光功率
> - 更近的辅光距离
> - 额外的轮廓光勾勒边缘

| 棕发 | 标准布光，适当轮廓光 | 注意头发高光不要过曝 |
| 金发 | 降低发丝光功率，避免过亮 | 使用蜂巢防止光线溢出 |

## 九、常见错误与修正

### 9.1 阴阳脸

**表现**：面部一侧亮、一侧暗，亮度差异明显

**原因**：
- 单侧主光过强
- 辅光位置不对或功率不足

**修正方法**：
```
修正前：主光 1:辅光 0 → 阴阳脸明显
修正后：主光 1:辅光 0.5 → 均匀过渡
```

1. 检查辅灯是否开启、位置是否正确
2. 增加辅灯功率或缩短辅灯距离
3. 使用白色反光板在被摄者阴影侧补光
4. 将主灯略微移向正面，减少侧光强度

---

### 9.2 鼻影过重

**表现**：鼻子下方出现明显深色阴影

**原因**：
- 主灯位置过高
- 光质过硬（无柔光附件）

**修正方法**：
```
        修正前                修正后
        主灯                   主灯
         ↓                      ↓
         ●过高                 ●适当高度
        ╱╲                    ╱  ╲
       ╱  ╲                  ╱    ╲
      ↓    ↓                ↓      ↓
     ●     ╲               ●───────●
     │      ╲                  浅阴影
     ↓       ↓
  深鼻影      (几乎无鼻影)
```

1. 降低主灯高度，与眼睛平齐或略高
2. 将主灯移近被摄者（增加光源面积）
3. 增大柔光箱尺寸
4. 添加正面辅灯（鳄鱼光下方灯光）

---

### 9.3 背景不均匀

**表现**：背景出现深浅不一的条纹或渐变

**原因**：
- 背景灯位置不当
- 主灯光线溢出到背景

**修正方法**：

1. **添加背景灯**：在背景正前方、距离约1.5米处添加一盏背景灯
2. **使用遮光旗板**：在主灯与背景之间放置黑色遮光板
3. **调整背景灯角度**：确保背景灯均匀照亮整个背景
4. **使用更宽的柔光箱**：减少主光方向性

---

### 9.4 曝光过度

**表现**：面部高光区域（如额头、鼻尖）过亮，细节丢失

**原因**：
- 闪光灯功率过高
- 光圈过大
- ISO过高

**修正方法**：
1. 降低闪光灯功率（-0.5EV至-1EV）
2. 收缩光圈（f/8→f/11）
3. 降低ISO（400→100）
4. 增加相机到被摄者的距离
5. 使用更大柔光箱柔化光线

---

### 9.5 眼神光问题

| 问题 | 原因 | 修正 |
|------|------|------|
| 无眼神光 | 正面无补光 | 在相机下方放置白色反光板 |
| 眼神光位置不对 | 反光板/灯光位置偏移 | 调整至瞳孔中心 |
| 眼神光过亮 | 补光过强 | 减弱补光或增加距离 |
| 眼神光形状不自然 | 光源形状问题 | 改用八角柔光箱（眼神光更圆） |

---

### 9.6 红眼

**表现**：瞳孔中出现红色反光

**原因**：
- 闪光灯直射眼睛
- 环境过暗，瞳孔放大

**修正方法**：
1. 使用跳闪或离机闪光，避免直射
2. 确保主环境有适当照明
3. 让被摄者拍摄前看亮处收缩瞳孔
4. 后期使用软件修正红眼

## 总结

### 布光核心要点

1. **均匀是基础**：面部无明显阴影是证件照的最低要求
