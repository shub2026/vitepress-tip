
## 环境和构建
确认本地已经安装Node.js
进入项目文件夹，安装NPM服务，启动开发环境，已经安装了就直接构建（预览本地构建服务）
构建静态文件

```
# npm安装
npm install

# 启动开发服务器
npm run docs:dev
 
# 构建静态
npm run docs:build

# 预览生产构建
npm run docs:preview
```
## 预览生产构建
如果已经执行过npm run docs:build生成了静态文件，可以使用以下命令本地预览生产版本。
这会启动一个本地静态Web服务（默认端口4173），以.vitepress/dist作为源文件。


- 首先，安装 vitepress。在终端中输入以下命令：npm add -D vitepress
- 使用安装向导 npx vitepress init 构建一个基本项目。
- 在 .vitepress 目录中配置你的网站。
- 在 .vitepress 目录之外的编写 Markdown 文件作为网站内容。
- 最后，使用 npm run docs:dev 启动并运行，大功告成！
