# Vitepress构建基本命令

## 环境构建
1. 确认本地已经安装Node.js
2. 进入项目文件夹，安装NPM服务，启动开发环境，已经安装了就直接构建（预览本地构建服务）
3. 构建静态文件

***拉取项目后，在项目内执行***
```sh
# npm安装
npm install

# 启动开发服务器
npm run docs:dev
 
# 构建静态
npm run docs:build

# 预览生产构建
npm run docs:preview
```
**预览生产构建**
如果已经执行过npm run docs:build生成了静态文件，可以使用以下命令本地预览生产版本。
这会启动一个本地静态Web服务（默认端口4173），以.vitepress/dist作为源文件。

### 快速命令速查

| 命令 | 作用 |
|------|------|
| `npm install` | 安装依赖 |
| `npm run docs:dev` | 启动本地开发服务器 |
| `npm run docs:build` | 构建生产版本 |
| `npm run docs:preview` | 预览构建结果 |
| `git remote -v`|查看远程仓库信息|

## 克隆拉取文件
适用于初次克隆拉取文件
```sh
# HTTP方式
 git clone https://gitee.com/shub77/Vitepress-tip
 git clone https://github.com/shub2026/Vitepress-tip

# SSH连接
 git clone git@gitee.com:shub77/vitepress-tip.git
```

## 关联仓库
```sh   
# SSH连接
 git remote add origin git@gitee.com:shub77/vitepress-tip.git
```

## 提交代码到 GitHub
1. 打开 VS Code 的终端（Ctrl + ~）
2. 依次运行这些命令：
```sh
git add .                            # 暂存所有改动
git commit -m "部署到 GitHub Pages"   # 创建一个提交
git push                             # 上传到 GitHub
```
---

## 网站更新
进入项目文件夹，顺序操作
```sh
# 1.拉取最新代码
git pull origin main

# 2.构建静态
npm run docs:build
```
