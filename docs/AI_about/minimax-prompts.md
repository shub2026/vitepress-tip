## 1. 模型概述

### 1.1 模型版本与特点

| 模型            | 发布时间   | 架构 | 总参数 | 激活参数 | 上下文窗口  | 核心能力               | 适用场景                     |
| --------------- | ---------- | ---- | ------ | -------- | ----------- | ---------------------- | ---------------------------- |
| **MiniMax M2.7** | 2026年4月  | MoE  | 456B   | ~45B     | **1M** tokens | 创意写作最强、性价比极高 | 内容创作、大规模调用、预算敏感 |
| **MiniMax M2.5** | 2025年     | MoE  | 456B   | ~45B     | 128K tokens  | 前代稳定版            | 通用轻量任务                 |

### 1.2 核心优势

✅ **创意写作国产最强**：创意写作评分 88 分，在文案、故事、营销内容方面表现突出  
✅ **极致性价比**：API 输入仅 ¥1 / 输出仅 ¥4（每百万 tokens），不到其他旗舰模型的 1/4  
✅ **百万级上下文**：M2.7 支持 1M tokens  
✅ **推理速度快**：仅 45B 激活参数，响应速度极快  
✅ **长文本处理强**：长文本理解评分 88，适合长篇内容处理

### 1.3 限制与注意事项

⚠️ **综合能力中等**：MMLU 82 / MATH 75，综合能力不及 DeepSeek V4 和 GLM-5.1  
⚠️ **编码能力一般**：编码评分 78，不适合高强度代码任务  
⚠️ **闭源**：不可自部署，仅通过 API 使用  
⚠️ **社区生态较小**：相比智谱、DeepSeek，社区资源和第三方工具较少

## 2. 提示词基础

### 2.1 MiniMax 提示词的基本结构

MiniMax 使用标准的 OpenAI 兼容 API 格式。

#### 使用 MiniMax API（推荐）

```python
from openai import OpenAI

client = OpenAI(
    api_key="YOUR_MINIMAX_API_KEY",
    base_url="https://api.minimax.chat/v1"
)

# 单次对话
response = client.chat.completions.create(
    model="minimax-m2.7",
    messages=[
        {"role": "system", "content": "你是一个创意写作助手。"},
        {"role": "user", "content": "写一篇关于春天的短文。"}
    ]
)

print(response.choices[0].message.content)
```

### 2.2 核心提示词技巧

#### ✅ 技巧1：创意写作最佳实践

MiniMax 的创意写作能力是其最大优势，适合需要文采和想象力的任务：

```python
prompt = """你是一位获奖的创意作家，擅长用文字构建画面感。

请写一篇微型小说，主题是「最后一个人类在火星上醒来」。

要求：
1. 字数：800-1000字
2. 要有强烈的画面感和情感共鸣
3. 使用丰富的比喻和拟人手法
4. 结尾要有反转或留白
5. 语言风格：诗意但不晦涩

额外要求：
- 开头用一句话抓住读者
- 在叙述中自然交代背景，不要大段说明
- 对话要自然，符合人物性格"""

response = client.chat.completions.create(
    model="minimax-m2.7",
    messages=[{"role": "user", "content": prompt}]
)
```

#### ✅ 技巧2：批量内容生成（性价比优势）

利用 MiniMax 的超低价格进行大规模内容生成：

```python
prompt = """你是一个电商文案写手。

请为以下10款产品各生成1条广告文案（每条30字以内）：

1. 无线蓝牙耳机（降噪版）
2. 智能手表（健康监测）
3. 便携式充电宝（20000mAh）
4. 机械键盘（青轴）
5. 电竞鼠标（RGB灯效）
6. 4K网络摄像头
7. 笔记本电脑支架（铝合金）
8. USB-C扩展坞（7合1）
9. 桌面拾音灯
10. 人体工学椅

每条文案格式：
【产品名】文案内容

风格：年轻化、有网感、适合短视频平台"""
```

#### ✅ 技巧3：营销内容创作

```python
prompt = """你是一个品牌营销专家。

请为一个新兴的精品咖啡品牌「晨露咖啡」创作以下内容：

1. **品牌Slogan**（5个备选）
2. **品牌故事**（300字，讲述品牌创立初心）
3. **小红书种草文案**（3条，每条200字，不同角度）

品牌调性：简约、自然、有品质感
目标受众：25-35岁都市白领，对生活品质有要求的咖啡爱好者
价格区间：中高端（单杯约8-15元）"""
```
