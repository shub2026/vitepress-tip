# 💰 模型接入汇总与售价

OpenClaw 理论上支持所有 OpenAI 格式的 API。以下是经过社区验证的接入方案汇总。

## 🔗 接入方案汇总
* **直接调用类**：DeepSeek, Moonshot, Zhipu AI (智谱), OpenAI, Anthropic。
* **中转平台类**：SiliconFlow (硅基流动), OpenRouter。
